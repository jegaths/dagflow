Welcome to Dagflow documentation!
===================================

**Dagflow** is an open source project aims to help create dag seamlessly and reliably. Dagflow API application is written in Python
and the frontend application is in ReactJs. Dagflow provides a simple and intuitive drag and drop UI to create dag and manage the dags. Dagflow can create new dags from the UI also dagflow can **import existing** dag files to dagflow UI.
Here you will find all the information about Dagflow project.

.. note::

   This project is under active development and currently in alpha state.
